## 📖 项目简介

智慧学习平台是一个功能完善、设计现代化的在线学习网站。项目采用纯前端技术栈（HTML5 + CSS3 + JavaScript）开发，包含首页、课程中心、学习计划、学习资源、个人中心、关于我们、联系我们等7个主要页面，实现了轮播图、表单验证、动态筛选等多种交互效果。

## ✨ 项目特色

- 🎨 **现代化UI设计**：采用渐变色彩、卡片式布局，视觉效果优秀
- 🚀 **丰富交互效果**：轮播图、表单验证、动态筛选、手风琴等
- 📱 **响应式布局**：完美适配桌面、平板、手机等不同设备
- 🎯 **功能完善**：涵盖在线教育平台的核心功能模块
- 💻 **代码规范**：遵循最佳实践，注释清晰，易于维护
- 🎓 **原创设计**：完全原创，无抄袭，具有实用价值

## 🛠️ 技术栈

- **HTML5**：语义化标签、表单验证
- **CSS3**：Grid布局、Flexbox布局、渐变、动画、响应式设计
- **JavaScript (ES6+)**：DOM操作、事件处理、表单验证、动态效果
- **Docker**：容器化部署（可选）
- **Nginx**：Web服务器（Docker模式）

## 🚀 快速开始

### 方法一：直接打开

1. 下载或克隆项目到本地
2. 直接双击 `index.html` 文件在浏览器中打开
3. 通过导航栏访问其他页面

### 方法二：使用Docker部署（推荐）

#### 前提条件
- 已安装 Docker（20.10+）
- 已安装 Docker Compose（1.29+）

#### 快速启动
```bash
# 1. 进入项目目录
cd label-2545

# 2. 启动服务
docker-compose up -d

# 3. 访问网站
浏览器打开：http://localhost:8080
```

#### 常用命令
```bash
# 停止服务
docker-compose down

# 重启服务
docker-compose restart

# 查看日志
docker-compose logs -f

# 查看运行状态
docker-compose ps

# 重新构建并启动
docker-compose up -d --build
```

#### Docker配置说明
- **端口映射**：8080:80（可在 docker-compose.yml 中修改）
- **开发模式**：文件挂载，修改即时生效
- **镜像大小**：约50MB（基于 nginx:alpine）
- **详细文档**：查看 [DOCKER.md](DOCKER.md)

### 方法三：使用本地服务器

```bash
# 使用Python
python -m http.server 8000

# 或使用Node.js
npx http-server

# 或使用Live Server（VSCode插件）
# 右键点击index.html，选择"Open with Live Server"
```

然后在浏览器访问：http://localhost:8000

## 🐳 Docker部署

### 配置文件
- **Dockerfile**：基于 nginx:alpine 的轻量级镜像
- **docker-compose.yml**：服务编排配置
- **nginx.conf**：Nginx服务器配置（Gzip压缩、缓存策略）

### 端口配置
默认端口：`8080`

修改端口（编辑 `docker-compose.yml`）:
```yaml
ports:
  - "你的端口:80"  # 例如: "3000:80"
```

### 开发模式 vs 生产模式

**开发模式**（文件挂载，修改即时生效）:
```bash
docker-compose up -d
```

**生产模式**（优化构建）:
```bash
docker-compose up -d --build
```

### 环境要求
- Docker 20.10+
- Docker Compose 1.29+
- 内存：最低512MB，推荐1GB
- 磁盘：最低100MB

完整的Docker部署文档请查看：[DOCKER.md](DOCKER.md)
